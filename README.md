## IEX Example Application

Visit it @ []

#### Version

```
  Ruby 2.7.2
  Rails 6.0.3.4
  NodeJS 12.18.3
  Yarn 1.22.4
```

#### Installation

```
  $ git clone git@github.com:avionschool/iex-app.git
  $ cd iex-app
```

#### Setup

```
  $ bundle install
  $ rake db:setup
```

#### Starting the application

```
  $ rails server
```

#### How to run the test suite

```
  $ rspec
```

#### How to run the Ruby Linter

```
	$ rubocop
```
